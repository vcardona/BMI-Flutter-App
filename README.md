# BMI-Flutter-Example

Ejercicio de creación de un BMI en el Bootcamp de desarrollo de aplicaciones moviles usando Flutter, en esta sección se aprendio:

* Como usar temas en Flutter.
* Como hacer un refactor de un Widget.
* Como crear Widgets personalizados.
* Uso del Slider Widget.
* Personalizar Widgets usnado Themes.
